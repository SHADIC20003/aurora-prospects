HIGH CVE-2026-46331

Out-of-bounds write in the Linux kernel traffic-control packet-editing action (act_pedit). Local unprivileged user can gain root by poisoning the cached copy of a setuid binary. Public exploit available.

DirtyClone — Linux Kernel (DirtyFrag Variant)

HIGH CVE-2026-43503 CVSS 8.8

New Linux kernel LPE in the DirtyFrag family. When the kernel copies a network packet internally, a safety flag is dropped. Local user can corrupt file-backed memory and gain root. First public exploit published June 25 by JFrog. Patch in mainline since May 21.
