HIGH CVSS 8.8
New DirtyFrag family variant — local privilege escalation via cloned network packets. Attacker with CAP_NET_ADMIN (via unprivileged user namespaces) corrupts file-backed memory, overwrites privileged binaries in memory. Fourth variant in this series.
