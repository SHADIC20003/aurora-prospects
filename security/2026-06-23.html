CVE-2026-54352 — Budibase Arbitrary File Read via Symlink Upload CRITICAL
CVE: CVE-2026-54352 (GHSA-w7mq-r738-x278)
Affected: Budibase (low-code platform)
Published: 2026-06-22
POST /api/pwa/process-zip accepts a builder-uploaded .zip file, extracts it, then serves extracted content. An attacker can embed a symlink in the zip pointing to an arbitrary file (e.g., /etc/passwd or database config), which is then read and served back. Allows arbitrary file read on the server.
